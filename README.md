## Overview
This repository documents the Microsoft 365 Identity Foundation build for Greenwood Accountants under the `christtech.co.uk` tenant.

The project implements a secure identity foundation including user provisioning, security groups, MFA via Conditional Access, Self-Service Password Reset, password protection, and least-privilege admin role assignments.

## Domain
- Tenant: `christtech.co.uk`
- Admin Centre: https://entra.microsoft.com
- User UPN format: `firstname.greenwood@christtech.co.uk`
